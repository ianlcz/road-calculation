#  RoadCalculation
Ce projet consiste en la création d'un algorithme permettant à l'utilisateur de connaître le temps à laquelle une livraison sera effectuée

# Utilisation
1. Cliquez sur le lien suivant [Allez sur l'application RoadCalculation](https://roadcalculation.yannlecoz.fr)
1. Une fois sur le site indiquez dans votre ville de départ ainsi que celle d'arrivée.
1. Cliquez sur le bouton **Calculez le temps du trajet**
